# Vision Stealer

Vision Stealer é uma ferramenta de builder com menu inicial para gerar e executar diferentes opções de stealer/utility.

## O que inclui

- Menu inicial com opções para:
  - Token Grabber
  - Discord Tools
  - Instalar dependências
- Builder com interface gráfica para gerar payloads
- Suporte a opções de coleta de dados e funcionalidades extras

## Estrutura do projeto

- `start.bat` — menu inicial do projeto
- `builder.pyw` — interface gráfica do builder
- `vision.py` — payload principal com as funcionalidades
- `requirements.txt` — dependências Python
- `tools/` — utilitários auxiliares
- `gui_images/` — imagens usadas na interface

## Requisitos

- Python instalado no sistema
- Acesso ao Windows
- Dependências listadas em `requirements.txt`

## Como usar

1. Abra o arquivo `start.bat`
2. Escolha uma opção:
   - `1` para Token Grabber
   - `2` para Discord Tools
   - `3` para instalar dependências
   - `4` para sair

## Instalação de dependências

Execute a opção `3` no menu inicial ou instale manualmente com:

```bash
pip install -r requirements.txt
```
